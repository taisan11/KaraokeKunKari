# Public Assets

このディレクトリには、MIDIファイルなどの公開アセットを配置します。

## MIDIファイルの配置

テスト用のMIDIファイルをこのディレクトリに配置することで、アプリケーションから読み込めます。

例：
- `sample.mid` - テスト用のサンプルMIDI
- `song1.mid` - 曲1
- `song2.mid` - 曲2

## 推奨されるMIDIファイルの形式

カラオケアプリに最適なMIDIファイルは以下の特徴を持ちます：

1. **メロディトラックが明確**: ボーカルメロディが1つのトラックに分離されている
2. **適度な音域**: 一般的な人間の歌唱音域（C3〜C6程度）
3. **クリアな音程**: 装飾音や和音が少ない、シンプルなメロディライン
4. **適度なテンポ**: 速すぎず遅すぎないテンポ

## MIDIファイルの入手方法

- 自分で作成（DAWソフトウェアなど）
- フリーのMIDIファイルサイトからダウンロード
- 既存の曲をMIDIに変換

## 注意事項

著作権のあるMIDIファイルを公開する場合は、適切な権利処理を行ってください。
